**N8N-ibex – Workflow Automation Templates**

This repository contains a collection of n8n workflow templates designed to automate real-world tasks across different domains. Each template solves a specific problem and can be customized or extended based on user needs.


**Included Workflows**

WhatsApp Workflow – AI chatbot for restaurants/hostels that handles client orders (supports text + voice).

AI Agent Chatbot – Connects with Gmail + Sheets for intelligent communication.

Gmail Label Workflow – Organize and label emails automatically.

Payroll System – Automates salary slip generation and notifications.

SEO Workflow – Helps manage SEO-related tasks.

Weather Workflow – Provides weather updates based on city input.


**Tech Stack**

n8n – Workflow automation

WhatsApp API / Gmail API / Other APIs – For integrations

AI / NLP – For chatbot and automation logic
